** Studies Manager API

Uma API robusta desenvolvida em **Node.js** e **Express** para gerenciar e monitorar o tempo dedicado aos estudos. Este projeto faz parte do meu portfólio de transição para **Full Stack Developer**.

## 🚀 Funcionalidades (CRUD)

- [x] **Cadastro de Estudos:** Registro de matéria e horas estudadas (com validação).
- [x] **Listagem:** Visualização de todos os registros salvos.
- [x] **Cálculo de Performance:** Rota que soma o total de horas dedicadas.
- [x] **Remoção:** Exclusão de registros específicos via ID.

## 🛠️ Tecnologias Utilizadas

- **Runtime:** Node.js
- **Framework:** Express
- **Ferramentas:** Nodemon (Desenvolvimento), Postman (Testes)
- **Linguagem:** JavaScript (ES6+)
